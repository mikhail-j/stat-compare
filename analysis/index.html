About this Project

Why?

This project presents Pokémon in a comparable form. The dataset used to form these statistics is called the "base statistics".

The Analysis

The maximum individual value (IVs) is 31. The maximum effort value (EVs) a Pokémon is 252. The nature of Pokémon can either be neutral, positive, or negative 10% of a specific stat.
These values can be used to calculate the actual stats of the specific Pokémon.

Base Stats were taken from Bulbapedia
